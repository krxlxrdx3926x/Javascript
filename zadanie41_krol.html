Pętla for

Pozwala na wykonywanie powtarzalnych zadań.
Konstrukcja

for (zainicjowanie_zmiennej; warunek_kończący_wykonywanie_pętli; zmiana_zmiennej) { kod który zostanie wykonany pewną ilość razy }

Pętla while z konrolowanym wejściem


pętla while najpierw sprawdza warunek, potem coś wykonuje, pętla może się nie wykonać.
Konstrukcja

while (wyrażenie_sprawdzające_zakończenie_pętli) { ...fragment kodu który będzie powtarzany... }
Schemat blokowy


Pętla do while z konrolowanym wyjściem


Pętla do...while zawsze wykona jedną iterację, zanim sprawdzi warunek. Zawsze więc wykona jakieś zadanie.
Konstrukcja

Do { ...fragment kodu który będzie powtarzany... } while (false)
