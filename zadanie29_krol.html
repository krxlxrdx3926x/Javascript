5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

This: pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu.
Oddzielanie pól i metod jest robione poprzez przecinek w obrębie jednego obiektu.

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Prototype: służy do definiowania metod i właściwości dla obiektu.
Właściwość prototype może być również wykorzystana do dodawania dodatkowych metod lub właściwości do istniejących obiektów. 

        
        function klient()
{
this.nazwisko='Bielski';
this.imie='Paweł';
}
klient.prototype.pisz_dane = function ()
 {
 document.write(this.nazwisko + ' ' + this.imie)
 }
klient.prototype.zawod = 'kierowca';
var osoba1 = new klient();
osoba1.pisz_dane();
